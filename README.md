# Python-Project

Global Fuel Price Analysis is a Python data analysis project focused on worldwide fuel price trends and a comparison of India's fuel prices with global averages.

## Project Structure

```text
Python-Project/
|-- README.md
|-- Global_fuelPrice_Analysis.py
|-- requirements.txt
|-- dataset/
|   `-- combined_fuel_dataset.xlsx
`-- outputs/
    |-- Bar_plot.png
    |-- Box_plot.png
    |-- Heatmap.png
    |-- Scatter_plot.png
    |-- Pieplot.png
    |-- Pairpot.png
    |-- Confusion_Matrix.png
    `-- India_vs_global_price.png
```

## Libraries Used

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- openpyxl

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the analysis script:

```bash
python Global_fuelPrice_Analysis.py
```

The generated visualizations are stored in the `outputs/` folder.
